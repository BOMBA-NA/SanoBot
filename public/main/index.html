Tutorial

Follow these steps to create and manage your bot:

1. Create or Login to Bot

Start by creating a new bot with your Facebook credentials or log in to an existing bot.

2. Set Bot Prefix

Choose a prefix character (like *, !, /, etc.) that will be used to trigger commands.

3. Add Admin ID

Set your Facebook ID as admin to access administrative commands.

4. Manage Commands

View available commands and their usage in the Commands section.
